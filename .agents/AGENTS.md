# Assistant Rules

- **Always** activate the project's virtual environment `chat-with-notes` before running any Python script or command in the terminal.
- Windows Command: `.\chat-with-notes\Scripts\activate` or directly reference `.\chat-with-notes\Scripts\python` / `.\chat-with-notes\Scripts\pip`.
- Linux/macOS Command: `source chat-with-notes/bin/activate` or reference `./chat-with-notes/bin/python` / `./chat-with-notes/bin/pip`.
